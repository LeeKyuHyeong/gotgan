이번 반영 (결정사항 1~4)
① 관리자 탭 없음 — 하단 탭은 홈/전체/이력/내정보 4개 유지. 가족장(owner) 권한은 ⑫ 내정보 안에 조건부 메뉴('가구 관리' 그룹)로. 멤버 화면에서는 이 그룹이 숨겨지고 '가구 나가기'만 노출.
② 분류 = 정해진 공통 목록(⑩ 칩 선택) + "분류 추가 요청하기" → 운영자가 ⑬⑭ 어드민에서 승인하면 공통 목록에 반영.
③ 위치 이모지 직접 선택 — ⑪ 위치 편집에 이모지 그리드 + 이름/순서/삭제.
④ 곧 만료 = D-3 이내 — 우유(D-1)·휴지(D-0)·비타민(D-2)만 뱃지. 계란(D-4)은 뱃지 없이 표기. 홈 '곧 만료 3개'로 재계산.

플랫폼 어드민(⑬⑭)은 사용자 앱과 완전히 분리된 별도 웹입니다(파란 테두리로 구분). 서버에서 role=SYSTEM_ADMIN으로 접근 통제, 운영자 본인만 사용. 일반 사용자 앱 번들에는 포함하지 않는 것을 권장합니다.

추가 반영 (2차)
⑤ 위치 편집 = 가족 구성원 누구나 — owner 전용 아님. ⑫ '위치 관리' 그룹에 '구성원 누구나' 표시.
⑥ 분류에 이모지 부여 — ⑩ 선택 칩·⑨ 입력·⑦ 아이템 행·⑭ 마스터 모두 이모지 표시. 어드민이 ⑬ 요청 승인 시 이모지를 지정.
⑦ 곧 만료 = 인앱 배지(v1) — 홈 상단 배너 + 하단 '홈' 탭에 숫자 배지(3). SMS/웹푸시/알림톡 등 외부 푸시는 v1 제외, 추후 결정.

아직 열려있는 것: 분류에 색상까지 줄지(지금은 이모지만) / 외부 푸시 채널(알림톡 등) 도입 시점 / 아이템 사진 첨부 여부.
